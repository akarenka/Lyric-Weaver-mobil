# ChronosImage AI Studio V6.2

**Image-to-Image • Storyboard • AI Prompt Studio**

------------------------------------------------------------------------

# English

## Introduction

ChronosImage AI Studio is a local AI image generation platform
supporting multiple AI providers.

### Supported Providers

-   Mock Generator (Offline Demo)
-   OpenAI GPT Image
-   Google Gemini Image (Nano Banana)

## Main Features

-   Text-to-Image
-   Image-to-Image
-   Storyboard Generator
-   Prompt Enhancement
-   Batch Generation
-   Prompt Library
-   Favorites
-   History Manager
-   Delete Single History
-   Force Clear All History
-   Metadata Export
-   Commercial Prompt Builder

## Installation

``` bash
install.bat
```

or

``` bash
pip install -r requirements.txt
```

## Start

``` bash
start_all.bat
```

or

``` bash
python app.py
```

Backend:

    http://127.0.0.1:7860

## API Keys

Edit `.env`

``` env
OPENAI_API_KEY=YOUR_API_KEY
GEMINI_API_KEY=YOUR_API_KEY
```

Restart the backend after saving.

# 日本語

## 概要

ChronosImage AI Studio は、複数の AI
プロバイダーに対応したローカル画像生成スタジオです。

### 対応プロバイダー

-   Mock Generator（オフラインデモ）
-   OpenAI GPT Image
-   Google Gemini Image（Nano Banana）

## 主な機能

-   テキストから画像生成
-   画像から画像生成（Image-to-Image）
-   ストーリーボード生成
-   プロンプト強化
-   バッチ生成
-   プロンプトライブラリ
-   お気に入り管理
-   履歴管理
-   個別履歴削除
-   履歴一括削除
-   メタデータ出力
-   商用レベルのプロンプト生成
